# SOSplugin
Rocket League Overlay

### It is an overlay for the video game Rocket League that uses a third party plugin to read real time data from the game so it can be manipulated and displayed.

#### Tracks:
Player Score <br />
Player Goals <br />
Player Assists <br />
Player Shots <br />
Player Saves <br />
Player Demos <br />
Player Boost <br />


Team Score <br />
Time Left <br />
Series Score <br />
Player Names <br />
Team Names <br />
